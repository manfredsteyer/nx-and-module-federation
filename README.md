# Example: Using Module Federation with Nx Monorepos

## Installing Dependencies

Use yarn (!) to opt-in into webpack5:

```
yarn
```

This is needed while the webpack5/CLI-integration is an experimental opt-in.

## Starting the Example

```
ng serve mfe1
ng serve shell
```
